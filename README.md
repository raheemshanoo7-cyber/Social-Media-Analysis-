🚀 Social Media Dashboard Analysis | Data-Driven Insights
Excited to share my latest Social Media Dashboard Analysis project! This interactive dashboard provides a comprehensive view of performance across multiple platforms, regions, and content types. It highlights key metrics such as total posts (5600), total views (5B+), total engagement (646M), and an average engagement rate of 15.28%.
📊 Key Insights:
Video content drives the highest engagement, followed by images and live streams
Platforms like TikTok, YouTube, and Instagram lead in overall engagement performance
Educational and product-related content categories generate the most user interaction
Regional analysis reveals strong engagement patterns across North America, Europe, and Asia
Monthly trends help identify peak engagement periods for better content planning
🛠️ Tools & Technologies Used:
Power BI (Dashboard development & data visualization)
Microsoft Excel (Data cleaning & preprocessing)
DAX (Data modeling & calculated measures)
Bing Maps integration (Geographic visualization)
This project helped me strengthen my skills in data storytelling, visualization design, and extracting actionable insights from complex datasets.
